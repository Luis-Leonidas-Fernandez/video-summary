# Monitoreo de recursos

## asitop

Monitor en tiempo real para Apple Silicon. Muestra CPU, GPU, ANE y memoria unificada.

**Abrir**
```bash
sudo asitop
```

**Cerrar**
```
q  ó  Ctrl + C
```

---

## btop

Monitor de procesos con RAM y CPU por proceso.

**Abrir**
```bash
btop
```

**Cerrar**
```
q  ó  Ctrl + C
```
